Short Project 1

1. There are 2 java files
2. The MergeLists.java is the class that has code for union, intersection, and set difference for sorted lists.
3. The Driver_MergeLists.java is the starter class for running this program.
4. The input is command line driven in the driver class. The number of elements for the lists can be changed inside the driver class by changing the value of n. Initially the value of n is 1 million.
